WHAT IT DOES
========================================

Automatically finds the REAL video URL from pages that
use "blob:" URLs (like SVT Play).

No more manually searching in DevTools Network tab!

========================================
FOR YOUR FRIEND - SIMPLE STEPS
========================================

STEP 1: GET THE CODE
--------------------
Option A - Download ZIP:
1. Go to: https://github.com/actualraptor/InGen-Youtube-Downloader
2. Click "Code" ‚Üí "Download ZIP"
3. Extract it

Option B - Git Clone (if you have Git):
git clone https://github.com/actualraptor/InGen-Youtube-Downloader
cd InGen-Youtube-Downloader
git checkout feature/auto-stream-detection


STEP 2: FIND A VIDEO
---------------------
1. Go to SVT Play website
2. Find any video you want
3. Copy the page URL (example):
   https://www.svtplay.se/video/abc123


STEP 3: FIRST TIME SETUP (One Time Only)
-----------------------------------------
1. Run the app (double-click run_gui.bat)
2. Paste the video URL
3. Click <F5> AUTO-DETECT STREAM
4. It will ask to install Playwright
5. Click YES
6. Wait 3-5 minutes (downloads ~200MB)
7. Close and restart the app


STEP 4: DOWNLOAD A VIDEO
-------------------------
1. Paste SVT Play URL in the box
2. Click <F5> AUTO-DETECT STREAM
3. Wait 30-60 seconds (watch the console)
4. URL will change to a .m3u8 stream
5. Click <F1> INITIATE to download!

========================================
HOW IT WORKS TECHNICALLY
========================================

1. Opens the SVT Play page in invisible Chromium browser
2. Watches ALL network traffic
3. Finds video stream URLs (.m3u8, .mpd, .mp4)
4. Picks the best quality stream
5. Updates your URL field
6. You download it normally
